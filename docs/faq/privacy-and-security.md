title: Privacy and Security
---

## Does LinguaX need cloud processing for core switching?

No. Core switching logic runs locally on your Mac.

## Does LinguaX require continuous internet access?

No for daily switching. Network is only needed for actions like downloads, updates, and license activation.

## Are custom scripts remote-executed?

No. AppleScript actions are executed locally.

## Why do some script actions ask for extra permissions?

macOS may request Automation permission when scripts control apps like System Events or Finder.

## What should I avoid sharing in support requests?

Do not share:

- passwords or secrets
- personal private content
- sensitive customer data in screenshots

## What is safe to include in a support report?

- LinguaX version
- macOS version
- exact reproduction steps
- exact error text
- sanitized screenshots/logs
