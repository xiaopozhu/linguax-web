---
title: Setup for Designers
---

This setup focuses on creative workflows that need stable shortcuts and predictable pointer behavior.

## Recommended Rule Baseline

Create app rules for:

- your main design tool
- browser
- communication/review tool

Keep defaults simple, then refine.

## Domain Rules to Add First

- design documentation
- internal review tools
- feedback platforms

## Mouse+ Focus

Start with high-impact controls:

- smooth scrolling
- preferred pointer behavior
- mouse button mapping if needed

Newer versions include extended mouse tilt button support (`MR/ML`).

## Validation Checklist

1. Switch between design app and browser.
2. Verify input behavior is correct in both.
3. Verify shortcut consistency.
4. Verify pointer and scrolling feel stable.
