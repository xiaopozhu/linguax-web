## Evaluation Criteria That Matter

- smooth scrolling quality in real apps
- side-button mapping reliability
- app-scoped override support
- low system overhead
- stability after sleep/wake and reconnect

## One-Day Test Protocol

Run each candidate through:

1. browser reading session (long pages)
2. editor + terminal switching
3. two mapped side-button actions
4. one reconnect cycle
