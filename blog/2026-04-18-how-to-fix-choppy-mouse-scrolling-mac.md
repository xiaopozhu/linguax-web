If your mouse feels choppy on macOS, you usually do not need a complicated setup.

This guide focuses on the shortest path to stable scrolling.

<!-- truncate -->

## Step 1: Establish a Clean Baseline

Start with one tool as source of truth for mouse behavior.

If multiple mouse tools are active, disable the others temporarily.

## Step 2: Enable Smooth Scrolling

In LinguaX Mouse+, enable smooth scrolling and test in:

- browser
- editor
- long document app

## Step 3: Tune One Parameter at a Time

Adjust only one setting, then test for 2-3 minutes:

- speed
- smoothness

Keep the change only if it improves consistency.

## Step 4: Add Per-App Overrides Only If Needed

If one app feels different, add app-scoped override there.

Avoid creating too many overrides early.

## Step 5: Validate in Real Workflow

Test in your normal context switching order for one full session.

If behavior stays stable, stop tuning.
